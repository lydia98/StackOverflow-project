Analysis about StackOverflow¶

Name: Zhiqi Fu¶

Introduction¶

I always use StackOverflow in my computer science courses. Whenever I have questions, I will search on the internet and most time, I could find the answers on StackOverflow. It is a really useful website for me. However, for now, most of my CS courses are about Java. I wondered if that website also popular in other computing languages. I used the dataset posts_questions to explore more about StackOverflow. The dataset collected questions posted on the StackOverflow and contained information like posted_date, tags, view_count...

Here are some questions that I want to answer:

  1. What computing language had been asked most frequently on Stackoverflow?

    • Which computing languages have the most questions number on StackOverflow?

    • How many view_counts are there for each computing languages?

  1. What is the development of Stackoverflow? Does it become more and more popular and well-known?

Analyzing the data¶

Question 1:¶

We look at the column 'tags' and want to see how many times each language appear in the tag. If that language is in the tag, that means the question is about that computing languages. Here, I focused on a few main computing languages that I have learned: Java, Python, SQL, Matlab.

In [11]:

sql = df[df['tags'].str.contains("sql", na=False)]

From the result, I learned that Java has the most posted questions among those four computing languages. Python and SQL have less, and Matlab has least questions posted. However, the number of questions posted did not represent a lot. I also want to see the average view counts of each question posted for each computing language. Therefore, I count the total view count and divide it by the number of questions posted to get the average.

In [12]:

The result is almost the same as the number of questions posted. Java has the most, Python and SQL have less and Matlab has least average view count. However, the difference between Java, Python, and SQL is not a lot.

From those two results, I could know that Java is a popular computing language that is discussed a lot on StackOverflow. Python and SQL are also popular and I could use StackOverflow to find the answers I want. However, there is not lots of information on StackOverflow for Matlab. It is a quite genuine result because I always found a great answer on StackOverflow, while I recalled when I learned Matlab, I rarely use StackOverflow. Instead, Mathworks is the website I could found the best solution.

Conclusion:¶

I always use StackOverflow but never have a chance to understand and develop more about this. From the above analysis, I found out that Java is still a popular language that is discussed a lot on StackOverflow. When students have questions about Java, Python, and SQL, they could use StackOverflow as a resource to solve their problems. However, StackOverflow is not the best place to find out the solution for Matlab's questions. What's more, StackOverflow becomes more and more popular from 2008 to 2018 that more questions are posted. Even the result of view counts of each year is out of my expectation that popular posts are in earlier years instead of recent years, I could not say the development of StackOverflow regressed. There might be some other factors caused that. Outliers could not represent a lot. In general, StackOverflow is still a fantastic website for people who learn Computer Science and becomes more and more popular.

Limitation:¶

  1. Since all the computing languages are showed in tags, instead of individually, I used "contains" method to find how many times those languages appeared. However, the result I found for Java will include both java and javascript, since javascript also contains "java". Also, I could apply this method to r language, since lots of words also contain "r". Another method, like "split" command could be used to solve this problem.

  1. The graph I drew for the view count of StackOverflow each year showed the outliers(high view count) clearly, but since all the lower view counts are gathered together, I could not have too much information for that part. Also, I could only find out the number of popular posts decreased, but I failed to think of a reason to explain that situation. This may be because from 2009 to 2013, computer science is still a new area and people did not have clear and absolutely correct answers to some basic questions, so they discussed a lot on StackOverflow. However, recently, people do not need to ask on StackOverflow for some basic questions. Most questions that posted on the StackOverflow are useful to a limited group of people. This is just my speculation and more information is needed to find out the reason.
